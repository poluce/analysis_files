# 自动更新 CLAUDE.md 文档

请执行以下步骤更新项目文档：

## 1. 分析最近的代码变更

查看最近 10 次 git 提交：
- 提取新增功能、bug 修复、重构等信息
- 识别是否有架构变化

## 2. 更新统计数据

自动统计并更新以下数据：
- 源文件总数（.cpp + .h）
- 代码行数（.cpp 文件）
- 算法实现数量
- 更新日期为当前日期

## 3. 更新"最近完成"列表

将最近的新功能添加到"最近完成 ✅"章节：
- 按时间倒序排列（最新的在最上面）
- 提取 git commit message 中的关键信息
- 格式化为项目符号列表

## 4. 检查需要补充的内容

检查是否有以下需要更新的部分：
- 新增的 UI 组件描述
- 新增的算法说明
- 架构设计的变化
- 交互功能的改进

## 5. 生成更新摘要

完成更新后，输出：
- 更新了哪些章节
- 添加了哪些新内容
- 建议用户关注的重要变化

注意：
- 保持文档风格一致（使用 Emoji、项目符号、代码块）
- 遵守代码注释规范（文档中可以使用 Emoji）
- 不要删除现有的重要内容，只添加和更新
